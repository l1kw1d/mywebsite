Bonjour Hydro...

Votre site web contient plusieurs failles, dont de nombreuses majeures..

Une liste rapide des types d'attaque et d'exploitations bien populaire chez les mal intentionner en ressort très rapidement...

  • Reflected XSS
  • Stored XSS
  • DOM XSS
  • Multiples Cross Site Scripting
  • Insecure/weak cross-domain policy
  • Abuse of Missing Content Security Policy
  • Abuse of XSS via CORS trust relationships

Le tout engendre plusieurs scénarios possibles d'attaque et d'abus des utilisateurs.

Les conséquences...

Vos employes, vos utilisateurs, vos clients sont tous des victimes car chacun utilise a leurs facon votre plateforme dans leur quotidien

L'usurpation de profile, l'exfiltration des données sensibles, collectes des données personnelles telles que les N.A.S, adresse physique, l'historique des déménagements, propriétaire d'unité, etc...

Abus de confiance

les conséquences des abus peuvent amener les victimes à ressentir de l'isolement, de la peur et de la méfiance, ce qui peut se traduire par des conséquences psychologiques à vie..

Dans l'immédiat

Vous avez des avis pour les utilisateurs sur votre site, indiquant d'actuels problèmes d'usurpation. The movement is done using the CSS property transition. I like using transition over @keyframes because when the toggle switches mid-animation there is no buggy-looking snap. Also, it's a bit more declarative, which is nice.
